Prometheus Integration

Export metrics in Prometheus format for external monitoring

Metrics Endpoint:
Sample Config:
scrape_configs:
  - job_name: 'claude-insight'
    static_configs:
      - targets: ['localhost:5000']
    metrics_path: '/metrics'
    scrape_interval: 15s
Available Metrics:
  • context_usage_percent - Context usage percentage
  • hooks_active - Enforcement hooks status (2 = both active)
  • policy_hits_total - Total policy hits counter
  • error_count_total - Total errors counter
  • session_count_total - Total sessions
